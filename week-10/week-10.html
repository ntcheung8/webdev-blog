Description

This week I learned about adding Google Chart graphs on w3schools. Learning this would allow me to create scatter charts, line charts, bar / column charts, area charts, pie charts, and more. I really like how simple the implementation for these graphs are, it's just a div in the html and the javascript code below. I also learned about some other javascript graphing techniques, like plotly and chart.js. Although, I think they serve basically the same purpose.

Below is an example of a Google Chart implementation with some random sample data.

Javascript for Google Charts

          
  google.charts.load('current',{packages:['corechart']});
  google.charts.setOnLoadCallback(drawChart);
  
  function drawChart() {
  // Set Data
  var data = google.visualization.arrayToDataTable([
    ['Price', 'Size'],
    [50,7],[60,8],[70,8],[80,9],[90,9],
    [100,9],[110,10],[120,11],
    [130,14],[140,14],[150,15]
  ]);
  // Set Options
  var options = {
    title: 'House Prices vs. Size',
    hAxis: {title: 'Square Meters'},
    vAxis: {title: 'Price in Millions'},
    legend: 'none'
  };
  // Draw
  var chart = new google.visualization.LineChart(document.getElementById('myChart'));
  chart.draw(data, options);
  }
