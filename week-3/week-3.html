Week 3 Description

This week I spent some time on W3Schools (5 short sections) and learned about how to embed video and audio to html. Adding autoplay, fullscreen, controls and volume was much easier than I expected, since it was just an extra setting on the <audio> tag Below is some sample code of how I added audio to this post.

The <audio> tag allows us to adjust features such as load, play, pause, duration and volume. In order to play audio after some event, you can use Document Object Model (DOM) events. I have not learned any Javascript yet so maybe that can go on the future to-do list.

Interestingly, one article noted that Chromium browsers do not allow autoplay alone, but muted autoplay is permitted.

Code Sample

    
    <figure>
        <h1>MP3 Audio</h1>
        <audio controls>
            <source src="audio/music.wav" type="audio/wav">
            Your browser does not support the audio element.
        </audio>
    </figure>
