Animation in CSS

October 7, 2021

Description

Today I spent some time on thoughtbot.com to learn how to do basic animation in CSS. Keyframes are the stages and styles of animation and animation properties assign keyframes to CSS elements. Here are a few things keyframes are composed of:

  • Name of animation
  • Stages of the animation
  • CSS Properties
Here are a couple things animation properties do:
  • Assigns keyframes to elements to animate
  • Defines how it's animation
And these two animation properties are required:
  • animation-name: defined in keyframes
  • animation-duration: duration in seconds (s) or milliseconds (ms)
Using these techniques we can make a few simple animations.

Here's a basic bounce in animation.

slooow

There are also some neat shortcuts we can use. The below code shows an example of using the shorthand animation syntax.

Code

            
    .bounce-in-repeat {
        height: 25vh;
        width: 20vw;
        text-align: center;
        background-color: aquamarine;
        animation:  bounceIn 1s ease-in-out 3s infinite;
        }

Output

fast non-stop
