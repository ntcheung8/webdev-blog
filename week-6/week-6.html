Rotating Cube

October 15, 2021

Description

This week I learned about creating a rotating cube on 3dtransforms.desandro.com. It was great to see how many of the basic principles I have learned could be applied to this more advanced animation. Although this is a little off topic, I learned how useful sites like coolers.co are for designing nice things. The way the position attribute is used was particularly cool to see, especially in a 3D object. An important note about the CSS, transform-style sets whether children of an element are in 2D (flat on screen) or 3D spaace.

One topic the article explored was un-fuzzing 3D transformed text. It turns out the 3D transforms can mess up text rendering. The fonts still have their old anti aliasing applied, and in order to fix this we move the 3D object inwards. This rotating cube was fun to learn about, especially because of how smooth the animations and transitions are!
