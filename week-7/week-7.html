This week I learned about scrolling effects on w3schools. Since smooth scrolling alone is simple, I also looked into some of the other tricks we can do with scrolling, like drawing and creating background gradients. The idea to look into scroll effects came from apple, since scrolling animations are all over their website. Although, some of those animations might be a little too complicated for me right now. Sites like freefrontend list many ways cool scrolling effects can be implemented. Maybe for my next site I should take inspiration from it.

Some Code for Drawing Triangle

      
  function myFunction() {
    var scrollpercent = (document.body.scrollTop 
      + document.documentElement.scrollTop) 
      / (document.documentElement.scrollHeight 
      - document.documentElement.clientHeight);
    var draw = length * scrollpercent;
    triangle.style.strokeDashoffset = length - draw;
  }
